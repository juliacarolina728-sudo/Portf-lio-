Radar Cidadão

O Radar Cidadão é um projeto desenvolvido com o objetivo de promover maior transparência e acessibilidade aos dados públicos, permitindo a análise do desempenho

de Deputados Federais em processo de reeleição por meio de dados da API da Câmara dos Deputados

O desenvolvimento foi realizado em equipe, sendo organizado em sprints.

Na Sprint 1, fui responsável pela prototipação da interface no Figma e pelo desenvolvimento do front-end utilizando HTML e Tailwind CSS.

Na Sprint 2, atuei na modelagem e implementação das estruturas do banco de dados utilizando Python, com apoio do Google Colab, além da configuração do ambiente em nuvem na AWS

Acesse o repositório no GitHub
